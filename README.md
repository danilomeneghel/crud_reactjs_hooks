# CRUD React

Projeto de um CRUD feito em ReactJS com API RESTful.

## Características

- CRUD
- API RESTful

## Requisitos
- NPM >= 6.4.1

## Tecnologias

- ReactJS

## Instalação

```
$ git clone https://github.com/danilomeneghel/crud-reactjs-hooks.git

$ cd crud-reactjs-hooks
$ npm install
$ npm start
´´´
