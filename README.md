# CRUD React

Projeto de um CRUD feito em ReactJS com API RESTful.

## Características

- CRUD
- API RESTful

## Requisitos
- NPM >= 6.4.1

## Tecnologias

- ReactJS

## Instalação

```
$ git clone https://github.com/danilomeneghel/crud_reactjs_hooks.git

$ cd crud_reactjs_hooks
$ npm install
$ npm start
```
